Velocity.js

A JavaScript animation engine

What it is

  • Animation engine
  • Has the same syntax as the most popular JavaScript library, jQuery
  • Allows user to control the DOM
  • Multiple animations sequentially
  • The best of jQuery and CSS transitions combined
  • Synchronizes the DOM â†’ tween stack to minimize layout thrashing
  • Cache values to minimize DOM querying.
  • Made by Julian Shapiro

Why it is needed

  • Better performance than jQuery, especially on mobile
  • Works everywhere, from IE8 and Andriod 2.3

What it does

  • Color animations
  • Transforms
  • Loops
  • Fade
  • Slide
  • SVG Support
  • Scroll events
